# 视频下载器

这是一个使用 `yt-dlp` 和 `Gradio` 构建的视频下载器。用户可以输入视频的 URL，程序将下载视频并提供预览。

## 功能

- 输入视频 URL 进行下载
- 检查缓存以避免重复下载
- 使用自定义文件名规则保存视频
- 提供视频预览功能

## 依赖

在运行此项目之前，请确保安装以下依赖项：

- Python 3.x
- yt-dlp
- gradio

您可以通过以下命令安装所需的库：

```
bash
pip install yt-dlp gradio
```
